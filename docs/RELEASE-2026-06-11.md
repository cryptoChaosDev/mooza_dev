# Релиз в PROD — 2026-06-11

Деплой: **web + api**. Миграции БД — **не нужны** (схема не менялась). Новых env — нет.
Базовая точка: `master 46c4765` → `dev` (10 фич-коммитов).

## Поток (лента)
1. **WYSIWYG-редактор постов** — жирный/курсив/зачёркнутый/списки/цитата/ссылка; HTML + санитайзер; обратная совместимость со старыми постами. *(client)*
2. **@-упоминания** (автокомплит, кликабельны) **+ вставка картинок по Ctrl+V/paste** (и iOS). *(client)*
3. **Быстрая сортировка + причёсанные фильтры** — дропдаун Новые/Популярные/Обсуждаемые, один ряд чипов, «Автор» в шторке. *(api+web)*
4. **Умная лента «Для вас»** — свежесть×вовлечённость + близость(друзья/подписки/избранное/артисты) + релевантность(город/жанр/сфера) + разнообразие авторов; **посты офиц. аккаунта всегда наверху**. *(api+web)*
5. **Ссылки в постах** — поле «ссылки» рендерится в ленте + голые URL кликабельны. *(client)*

## Профиль
6. **Уникальность ника при смене** — живая проверка + не закрывать редактор при ошибке. *(client)*
7. **Имя/Фамилию нельзя стереть** — обязательны (клиент + серверная валидация). *(api+web)*
8. **Фиолетовое кольцо вокруг аватара** по проценту заполнения профиля. *(client)*
9. **Редизайн портфолио** — аудиоплеер, иконки по типу файла, сетка фото, лимит 10 МБ на фото/доки. *(api+web)*

## Pro
10. **Срок подписки виден пользователю** + **админ может аннулировать Pro** (карточка + вкладка донатов). *(api+web)*

## Каталог
11. Вкладка **«Каталог» → «Услуги»**. *(client)*

## Платформа (сквозное)
12. **Тосты-ошибки на всех CRUD** — каждая add/edit/delete показывает причину сбоя. *(client)*

## Шаги релиза
1. Бэкап PROD-БД (`pg_dump` → `/root/mooza_prerelease_*.sql`).
2. `merge dev → master`, push.
3. PROD: `git pull master` → `docker compose build web api` → `up -d`.
4. Smoke (home/api/feed/smart/каталог).
5. **Разовая починка данных:** старые битые имена файлов в портфолио (UTF-8 mojibake) — безопасный скрипт `fixIfGarbled` (сухой прогон → подтверждение → применение). На DEV уже применено (5 файлов).
