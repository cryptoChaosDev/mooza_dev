- [ ] Кнопка "Оформить сделку" на ServicePage → форма создания сделки (название, услуга, стоимость, сроки, кол-во правок, исполнитель)
- [ ] Кнопка "Оформить сделку" в ConnectionViewModal (для ACCEPTED-связи)

- [ ] Сценарий отмены с разными штрафами (до/после оплаты)
- [ ] Автоматическое создание Connection при завершении сделки
- [ ] Счётчик сделок на профиле (сейчас всегда "0")
- [ ] Блок "Мои сделки" на ProfilePage — слайдер с активными сделками
