### Сделки (Deal Page + DealsPage)
- [x] **Тип А (process)**: PENDING → AWAITING_PAYMENT → IN_PROGRESS → REVIEW ↔ REVISION → COMPLETED
- [x] **Тип Б (event)**: PENDING → AWAITING_PAYMENT → AWAITING_EVENT → AWAITING_CONFIRMATION → COMPLETED (с депозитом и датой события)
- [x] Кнопка «Дублировать» сделку
- [x] **Редактирование условий после принятия** (Срок/Правки) с согласованием второй стороной (DealEditRequest)
- [x] **Автоматические таймауты** (cron каждые 60 сек):
  - IN_PROGRESS + истёкший deadline → CANCELLED
  - REVIEW + истёкший acceptDeadline → COMPLETED
  - AWAITING_EVENT + наступившая дата → AWAITING_CONFIRMATION
  - AWAITING_CONFIRMATION + истёкшие 3 дня → COMPLETED
- [x] Авто-чат «Деловые», авто-связь при завершении
- [x] Уведомления на каждый шаг (10+ типов)
- [x] «Оценить взаимодействие» (10 звёзд) на завершённой сделке
